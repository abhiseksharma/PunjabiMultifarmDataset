# Punjabi Multifarm Dataset Extension

This repository contains the Punjabi extension of the multilingual ontology matching benchmark derived from the Ontology Alignment Evaluation Initiative (OAEI) Multifarm track. The dataset has been developed to support research in multilingual ontology matching, low-resource language processing, and culturally-aware semantic alignment.

The primary objective of this work is to extend multilingual ontology matching evaluation beyond high-resource languages by introducing Punjabi, a low-resource language written primarily in the Gurmukhi script. The dataset enables researchers to evaluate the robustness and generalization capabilities of ontology matching systems under linguistically diverse and resource-constrained settings.

# Overview

The repository includes:

Punjabi ontology translations
Reference alignment files
Dataset generation scripts
Preprocessing and transformation utilities
Evaluation resources
Documentation for reproducing the dataset generation pipeline

The proposed framework is designed to be modular and reusable, allowing researchers to generate similar multilingual ontology matching datasets for other low-resource languages with minimal modifications.

# Relationship with Previous Work

This work extends our earlier framework developed for the Hindi multilingual ontology matching dataset. The source code and dataset generation pipeline used in this repository are adapted and extended from that earlier work.

The Punjabi extension introduces additional challenges related to:

low-resource lexical availability,
script handling,
transliteration ambiguity,
multilingual semantic alignment,
and culturally dependent terminology.

By building upon the previously developed Hindi framework, this repository demonstrates the extensibility of the proposed multilingual dataset generation approach toward other underrepresented languages.

# Dataset Description

The dataset is based on ontology pairs from the OAEI Multifarm benchmark and extends them with Punjabi translations and multilingual alignments.

The dataset can be used for:

multilingual ontology matching,
low-resource semantic alignment,
multilingual knowledge graph research,
cross-lingual semantic interoperability,
ontology evaluation benchmarking,
and culturally-aware semantic AI research.

# Framework Reusability

The provided framework is language-independent and can be adapted to generate ontology matching datasets for other languages by modifying:

translation resources,
language mappings,
lexical processing modules,
and alignment generation configurations.

Researchers are encouraged to extend the framework for additional low-resource and culturally diverse languages.
